# Rutas de Imágenes – Proyectos del Portafolio

Guarda las capturas de pantalla de cada proyecto en la siguiente carpeta del repositorio:

```
PORTAFOLIO/
└── public/
    └── images/
        └── projects/
            ├── game-paradise.png         ← Captura de Game Paradise
            ├── historia-mundiales.png    ← Captura de Historia de los Mundiales
            ├── movie-box.png             ← Captura de Movie Box
            └── portafolio.png           ← Captura del Portafolio Profesional
```

## Especificaciones recomendadas

| Campo        | Valor recomendado                     |
|--------------|---------------------------------------|
| Formato      | `.png` o `.webp` (mejor compresión)   |
| Resolución   | 1280 × 720 px mínimo (16:9)           |
| Peso máximo  | 500 KB por imagen                     |
| Tipo         | Captura de pantalla o mockup del proyecto |

## Notas

- Si la imagen no existe o no carga, la tarjeta muestra automáticamente el emoji del proyecto como placeholder — no se rompe el diseño.
- Puedes usar capturas directas del navegador, mockups en dispositivos o imágenes de la interfaz principal.
- Los nombres de archivo **deben coincidir exactamente** con los indicados arriba (respeta minúsculas y guiones).
