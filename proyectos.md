### Game Paradise – Plataforma Web Full Stack (UANL) 2025

**Frontend Developer**

- Desarrollo de plataforma web de compra y venta de videojuegos tipo marketplace.
- Frontend desarrollado con React, diseño responsivo y enfoque en experiencia de usuario.
- Backend con Node.js y Express, implementación de API REST.
- Uso de MongoDB para gestión de usuarios, juegos, carritos y órdenes.
- Implementación de autenticación, roles (cliente / vendedor), carrito de compras y sistema de pedidos.
- Integración de comentarios, calificaciones y favoritos.
- Manejo de WebSockets para funcionalidades en tiempo real.
- Control de versiones con Git y GitHub.
- **Tecnologías:** React, JavaScript, Node.js, Express, MongoDB, APIs REST, WebSockets, Git.
- **Proyecto:** https://github.com/Gamer-red/Mi_prograweb2

### Movie Box – Plataforma Web Full Stack (UANL) 2026

**Full Stack Web Developer**

- Desarrollo de plataforma web móvil (Marketplace) enfocada en la compra y venta de películas.
- Frontend desarrollado con React, diseño responsivo y animaciones fluidas con Framer Motion.
- Backend construido con Node.js y Express mediante la implementación de una API REST.
- Uso de MongoDB para la gestión de usuarios, películas, carritos de compra, valoraciones y órdenes.
- Implementación de sistema multi-rol independiente para Clientes y Vendedores.
- Integración de un Dashboard de negocio para vendedores con reporte de ventas y filtros avanzados por fechas.
- Configuración de seguridad mediante el hashing de contraseñas con Bcrypt y autenticación con tokens JWT.
- Gestión de carga de archivos multimedia e imágenes de perfil/productos con Multer.
- Control de versiones con Git y GitHub.
- **Tecnologías:** React, JavaScript, Node.js, Express, MongoDB, APIs REST, JWT, Bcrypt, Multer, Framer Motion, Git.
- **Proyecto:** https://github.com/CATCATUWU/Movie-Box
